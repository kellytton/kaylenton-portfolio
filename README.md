# Kaylen Ton's Portfolio

A personal portfolio website for Kaylen Ton, a Data Scientist and Analyst based in Tampa, FL.

## Tech Stack

- ReactJS
- Vite
- Material UI (MUI)

## Getting Started

### Prerequisites

- Node.js 20.19+ or 22.12+

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Project Structure

```
src/
├── assets/          # Images, resume, project slides
├── components/      # React components
│   ├── navbar/
│   ├── hero/
│   ├── about/
│   ├── skills/
│   ├── projects/
│   └── footer/
├── data/            # JSON data files
├── App.jsx
└── main.jsx
```
